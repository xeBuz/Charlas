¿Qué es GNU/Linux?*

*Si, pueden decirle sólo Linux, por más que Stallman se enoje

GNU

Es un proyecto que nace con un principio muy claro. Es software debe ser libre.



Linux

Es el sistema operativo en sí (sería el equivalente a Windows), lo que ayuda al hardware, a las aplicaciones ya al usuario interactuar.

Software Libre...


  • Establece que el software debe ser Libre, pero no confundirlo con Gratis, va más allá de eso.
    • (Free as a bird, not as in free beer)
  • Libre quiere decir que se distribuya el código fuente.
  • Compartir está bueno.
  • Usar software privativo, si les sirve, no está mal.

    • Podemos comparar el software libre con una receta, al compartirla, otros pueden cocinar lo mismo e incluso mejorarla, dándote crédito por lo que hiciste inicialmente. Y todos ganan, y compramos menos Pepitos y Toddys.
