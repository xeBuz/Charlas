Qué es GNU/Linux?*

*Si, pueden decirle sólo Linux

GNU

Es un proyecto que nace con un principio muy claro. Es software debe ser libre.



Linux

Es el sistema operativo en sí (Sería el equivalente a Windows o iOS), lo que ayuda al hardware, a las aplicaciones ya al usuario interactuar.

Software Libre...


  • Significa "GNU is Not Unix".
  • Establece que el software debe ser Libre.
    • (Free a bird, not as in free beer)
  • Libre quiere decir que se distribuya el código fuente. Podemos compararlo con una receta, al compartirla, otros pueden cocinar lo mismo e incluso mejorarla, dándote crédito por lo que hiciste inicialmente
  • Compartir está bueno.
  • Usar software privativo, si les sirve, no está mal. *muere dilapidado por fanboys*
