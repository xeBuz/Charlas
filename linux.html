Qué es GNU/Linux?*

*Si, pueden decirle sólo Linux

GNU

Es un proyecto que nace con un principio muy claro. Es software debe ser libre.



Linux

Es el núcleo, el sistema operativo en sí, lo que ayuda al hardware, a las aplicaciones ya al usuario interactuar.

Linux


  • Linux es el Sistema Operativo.
  • Sería el equivalente a Windows o iOS.
  • Es desarrollado por las mismas personas que lo usan, no hay una empresa que lo haga.
  • Existen muchas opciones, te da cientos de opciones para cada uso.
